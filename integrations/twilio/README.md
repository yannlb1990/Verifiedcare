# Twilio Integration

## Status: Not Configured

## Purpose
- SMS notifications for bookings
- Phone verification
- Appointment reminders

## Required Credentials
- TWILIO_ACCOUNT_SID
- TWILIO_AUTH_TOKEN
- TWILIO_PHONE_NUMBER

## Setup
1. Create Twilio account at twilio.com
2. Get credentials from Console
3. Purchase a phone number
4. Add to .env file
