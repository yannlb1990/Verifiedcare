# AWS S3 Integration

## Status: Not Configured

## Purpose
- Store provider documents
- Store compliance certificates
- Store user avatars

## Required Credentials
- AWS_ACCESS_KEY_ID
- AWS_SECRET_ACCESS_KEY
- AWS_REGION
- AWS_S3_BUCKET

## Setup
1. Create AWS account
2. Create S3 bucket
3. Create IAM user with S3 access
4. Add credentials to .env file
