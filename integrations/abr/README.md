# ABR Integration

## Status: Not Configured

## Purpose
- Verify provider ABN
- Check business registration status
- Validate GST registration

## Required Credentials
- ABR_GUID

## Setup
1. Register at abr.business.gov.au/Tools/WebServices
2. Request web services access
3. Get GUID from confirmation email
4. Add to .env file

## API Endpoint
- https://abr.business.gov.au/abrxmlsearch/AbrXmlSearch.asmx
